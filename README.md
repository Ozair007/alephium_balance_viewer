# Alephium Balance Viewer

A React + TypeScript application for viewing Alephium (ALPH) wallet balances.

## Features

- View ALPH balance and balance hints for any Alephium address
- Clean, modern UI built with React and Shadcn/UI
- TypeScript for type safety

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run format` - Format code with Prettier
- `npm test` - Run tests in watch mode
- `npm run test:run` - Run tests once

## Bash Script

A bash script for fetching ALPH balances via CLI is available in `src/scripts/get-balance.sh`.
